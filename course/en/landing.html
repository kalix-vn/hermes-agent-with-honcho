Architecture

Two halves of an assistant

Not a chatbot. One half knows how to act, the other helps it understand and remember you.

Hermes · Action

The hands & the skillset

The acting brain: calls the LLM, uses tools, runs scheduled tasks, and serves a dashboard to chat.

  • Skills — loaded on demand (progressive disclosure)
  • Crons — run work on a schedule, proactive
  • Soul — customizable personality & tone
  • Self-improvement loop — better the more you use it
Honcho · Memory

Memory & understanding

A self-hosted memory layer: not just storing text, it models you and reasons in the background.

  • Peer · Session · Workspace — the memory structure
  • Deriver — asynchronous reasoning worker
  • Representation — a living portrait of you
  • Dialectic — reasoning, not just lookup
