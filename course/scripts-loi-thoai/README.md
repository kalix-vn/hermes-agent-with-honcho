# 🎙️ Kịch bản lời thoại (word-for-word)

Thư mục này chứa **lời thoại đầy đủ để đọc thẳng trước camera** cho một số module mẫu — khác với
file module (là *kịch bản dạy học*, gồm bước + lệnh + checklist). Ở đây là **VO (voice-over) từng
câu**, kèm mốc thời gian và cue màn hình, để bạn quay/lồng tiếng ngay.

## Quy ước trong file lời thoại
- `[00:00]` — mốc thời gian gợi ý.
- `【MÀN HÌNH: ...】` — cắt cảnh / cần hiện gì trên màn hình.
- `【B-ROLL: ...】` — cảnh chèn minh họa.
- `【THAO TÁC: ...】` — việc cần làm live trước camera.
- Chữ thường in nghiêng _(…)_ = ghi chú cho người quay, **không đọc**.
- Còn lại: đọc y như viết (đã tối ưu cho văn nói tiếng Việt).
